HTML Tags Used in This Project

HTML Tag Purpose
<html>Defines the HTML document.
<head>Contains page information and metadata.
<title>Displays the page title in the browser.
<meta>Provides metadata such as character encoding and responsive settings.
<link>Connects the external CSS stylesheet.
<body>Contains all visible webpage content.
<header>Creates the website header.
<nav>Creates the navigation menu.
<ul>Defines an unordered list.
<li>Represents a list item.
<a>Creates hyperlinks between pages.
<section>Groups related webpage content.
<h1> - <h3>Defines headings.
<p>Displays paragraphs.
<img>Displays images.
<audio>Embeds audio files.
<video>Embeds video files.
<table>Creates tables.
<tr>Defines a table row.
<th>Creates table headings.
<td>Creates table cells.
<form>Creates a registration form.
<input>Accepts user input.
<select>Creates a drop-down list.
<option>Defines options inside a drop-down list.
<footer>Creates the website footer.

Definition of Each Page

Page Description
Home Introduces the website owner and explains the purpose of the portfolio website.
Images Displays six images arranged in an HTML table.
Audios Displays six audio files using the HTML5 audio player.
Videos Displays six videos using the HTML5 video player.
Registration Contains a complete HTML5 registration form with different input types.
Table Displays HTML tags used in the project and explains the purpose of each page.

Example of HTML Code Used

Example Purpose
<img src="images/image1.jpg" alt="Image"> Displays an image from the images folder.
<audio controls> Creates an HTML5 audio player.
<video controls> Creates an HTML5 video player.
<input type="email"> Creates an email input field.
<a href="images.html">Images</a> Creates a hyperlink to another webpage.
